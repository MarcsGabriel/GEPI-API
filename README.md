## Steps to execute:

> clone o repositório: https://github.com/MarcsGabriel/GEPI-API

~~~node
  npm init
  npx sequelize db:migrate
  npm start
~~~

## Steps to use:

> open your rest request service: **insomnia**, **apidog**, **postman**, **swagger**

> imports a collection: "collection_requests.json"
